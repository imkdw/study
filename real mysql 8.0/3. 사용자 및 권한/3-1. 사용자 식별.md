# 사용자 식별

- MySQL은 사용자 식별시 아이디랑 호스트를 같이 명시해야함

### 로컬 호스트에서 접속가능한 유저

```
'user_id'@'127.0.0.1'
```

### 모든 호스트에서 접속가능한 유저

- %는 모든 호스트를 의미함

```
'user_id'@'%'
```

### 범위 접근

- `user_id'@'192.168.0.1'`, `user_id'@'%'` 2개의 사용자가 있다고 가정
- MySQL은 권한/사용자 정보에 대해서 항상 범위가 좁은걸 먼저 선택함
- 위 2개의 계정에서 좁은범위는 `user_id'@'192.168.0.1'`으로, 해당 계정으로 인증을 진행함
