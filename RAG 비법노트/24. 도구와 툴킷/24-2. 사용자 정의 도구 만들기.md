# 사용자 정의 도구 만들기
- `langchain.tools` 모듈의 `@tool` 데코레이터를 활용해서 일반 함수를 도구로 등록함
- 원하는 로직을 함수로 구현한뒤 `@tool`을 붙이면 에이전트가 해당 함수를 하나의 도구처럼 사용이 가능함
- 이 때 LLM이 어떤 상황에 이러한 함수를 사용할지 `docstring`을 통해 설명해주는게 중요함
- 설명은 LLM이 잘 이해할 수 있게 영어로 작성하는게 좋음

<br>

# 도구 작성 예시
```python
from langchain.tools import tool


@tool
def add_numbers(a: int, b: int) -> int:
    """Add two numbers"""
    return a + b


@tool
def multiply_numbers(a: int, b: int) -> int:
    """Multiply two numbers"""
    return a * b


add_numbers.invoke({"a": 3, "b": 4})

multiply_numbers.invoke({"a": 3, "b": 4})
```

# 키워드 기반으로 구글 뉴스 기사 검색하는 도구 만들기

from langchain_teddynote.tools import GoogleNews
from langchain.tools import tool
from typing import List, Dict


# 키워드로 뉴스를 검색하는 도구
@tool
def search_keyword(query: str) -> List[Dict[str, str]]:
    """Look up news by keyword"""
    print(query)
    news_tool = GoogleNews()
    return news_tool.search_by_keyword(query, k=5)
