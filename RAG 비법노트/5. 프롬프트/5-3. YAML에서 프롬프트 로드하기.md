# YAML에서 프롬프트 로드하기
- 프롬프트가 길어지면 코드 내의 문자열이 너무 길어서 관리하기 힘듬
- 외부 파일로 빼서 관리하면 유용함

```python
from langchain_openai import ChatOpenAI
from dotenv import load_dotenv
from langchain_core.prompts import load_prompt
from pathlib import Path

load_dotenv()

llm = ChatOpenAI(model="gpt-4o-mini")

# 현재 파일의 디렉토리를 기준으로 절대 경로 생성
current_dir = Path(__file__).parent
yaml_path = current_dir / "capital.yaml"

prompt = load_prompt(str(yaml_path), encoding="utf-8")

# input_variables=['country'] input_types={} partial_variables={} template='{country}의 수도에 대해서 알려주세요.\n수도의 특징을 다음의 양식에 맞게 정리해 주세요.\n300자 내외로 작성해 주세요.\n한글로 작성해 주세요.\n----\n[양식]\n1. 면적\n2. 인구\n3. 역사적 장소\n4. 특산품\n\n#Answer:\n'
print(prompt)

"""
한국의 수도에 대해서 알려주세요.
수도의 특징을 다음의 양식에 맞게 정리해 주세요.
300자 내외로 작성해 주세요.
한글로 작성해 주세요.
----
[양식]
1. 면적
2. 인구
3. 역사적 장소
4. 특산품

#Answer:
"""
print(prompt.format(country="한국"))
```
