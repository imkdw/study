# Ollama 모델을 사용한 RAG
- 완전히 폐쇠된 환경에서 RAG 시스템을 만들려면 허깅페이스나 Ollama 모델 사용이 가능함
- 하지만 고성능 GPT 모델과 비교하면 현저히 떨어지므로 주의필요

<br>

### Ollama 사용하기
```bash
# 모델 다운로드
ollama pull gpt-oss:20b
```

```python
from langchain_ollama import ChatOllama

llm = ChatOllama(model="gpt-oss:20b")

result = llm.invoke("메이플스토리가 뭔가요?")

"""
**메이플스토리(MapleStory)**는 한국의 대형 게임 회사인 **넥슨(Nexon)**이 개발·운영하는 **2D 횡스크롤 액션 MMORPG**입니다.  

| 항목          | 내용                                                                                                                                                                                                                                                                                                                                                                               |
| ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **출시일**    | 2003년 10월 29일 (한국), 이후 2005년 미국과 2006년 일본 등 전 세계로 확장                                                                                                                                                                                                                                                                                                          |
| **플랫폼**    | PC(Windows), 모바일(안드로이드·iOS), 일부 모바일/콘솔 버전                                                                                                                                                                                                                                                                                                                         |
| **게임 방식** | 픽셀 아트 스타일의 캐릭터를 조작해 퀘스트를 수행하고, 전투 시 화면을 좌우로 스크롤하며 무기를 댕겨서 적을 물리칩니다.                                                                                                                                                                                                                                                              |
| **전투**      | 실시간 액션 전투(레벨에 따라 각기 다른 스킬과 무기), 타임링과 콤보가 핵심                                                                                                                                                                                                                                                                                                          |
| **게임 특징** | ① **캐릭터 직업**: 총 20개의 직업과 170여개의 서브클래스(버프, 파티 직업 등), 직업마다 독특한 스킬 구성을 갖춤 ② **커뮤니티와 파티**: 4~6명 파티를 구성해 레이드 던전 등에 도전 ③ **경제와 아이템**: 무한으로 생성되는 재료와 드롭 아이템, 캐시피(현금 아이템)을 통한 아이템 거래 ④ **콘텐츠**: 정규 월드, 파티 던전, 크리에이션(아이템 제작), 이벤트, PvPvE 등 다양한 콘텐츠 제공 |

### 메이플스토리의 성장과 영향
- **무료(Free‑to‑Play)**: 처음부터 아이템 구매 없이 무료로 시작할 수 있어 유저 층이 폭넓음  
- **문화·경제적 현상**: 게임 내 판매 아이템, 스킨, 캐시피가 활성화된 경제와 게임 내 토픽이 사회적·문화적 이슈가 됨  
- **콘텐츠 업데이트**: 매달 새로운 서포트나 시즌(Seasonal System)을 통해 캐릭터 성장, 스토리 추가  
- **메이플스토리: M**: 2015년 모바일 전용 버전으로 전 세계 모바일 게이머까지 확대

- **확장·시리즈**: 메이플스토리와 연계된 파생 게임(예: 메이플버틀, 메이플스톰) 및 글로벌 서비스가 지속적으로 공개됩니다.  

> **요약**: 메이플스토리는 2003년부터 시작된 2D 횡스크롤 액션형 MMORPG로, 한국에서 처음으로 큰 인기를 끌었고 이후 전 세계적으로 많은 유저를 확보한 게임입니다. 게임 내부는 캐스터, 전사, 궁수 등 다양한 직업과 무한히 확장될 수 있는 아이템, 실시간 액션 전투가 특징이며, 무료 게임으로서 누구나 쉽게 접할 수 있습니다.
```
