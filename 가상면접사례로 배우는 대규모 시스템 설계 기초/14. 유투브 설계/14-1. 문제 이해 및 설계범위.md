# 필요한 기능

- 빠른 비디오 업로드
- 원활한 비디오 재생
- 재생 품질 선택 기능
- 낮은 인프라 비용
- 높은 가용성과 규모 확장성, 안정성
- 지원 클라이언트 : 모바일 앱, 웹브라우저, 스마트 TV

<br>

# 개략적 규모

- DAU : 5,000,000
- 1명의 사용자는 평균 하루에 5개의 비디오를 시청
- 10% 사용자가 하루에 1비디오 업로드
- 비디오의 평균 크기는 300MB
- 비디오 저장을 위한 하루당 용량 = `500만 * 10% * 300MB = 150TB`
- CDN 비용
  - AWS CloudFront를 기준 모든 트래픽이 미국에서 발생한다면 1GB당 0.02의 요금 발생
  - 매일 `5백만 * 5비디오 * 0.3GB * $0.02 = $150,000` 발생
